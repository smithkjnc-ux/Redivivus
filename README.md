**Redivivus keeps your AI coding organized so nothing gets lost.**

If you're using an AI (like Windsurf, Cursor, Claude, or ChatGPT) to help you build something, Redivivus is the tool that keeps everything on track. It saves your notes between sessions, keeps your files tidy, and gives your AI a clear picture of your project every time you start.

Think of it like a job site clipboard — it holds the plan, the checklist, and the notes so nothing falls through the cracks.

---

## What it does

- 📋 **Remembers your project** — describe what you're building once, and Redivivus keeps it on file so your AI always has context
- ▶️ **Tracks your sessions** — log what you worked on, what's done, and what's next every time you sit down to code
- 🔍 **Scans your project** — finds messy files, unfinished TODOs, and things that need attention
- 🔧 **Cleans up your code** — adds organization notes to your files so the AI can navigate them better
- 💾 **Saves reusable pieces** — stores useful code snippets in a Vault so you can reuse them in future projects

---

## Getting started

1. Install Redivivus from the VS Code marketplace
2. Open your project folder in VS Code / Windsurf / Cursor
3. Click the **Redivivus icon** in the left sidebar
4. Click **"Set Up My Project"** and answer 5 quick questions
5. Click **"Start Working"** before each session — **"Done for Now"** when you stop

That's it. Redivivus handles the rest.

---

## Who is this for?

Anyone building something with AI help — even if you've never written code before. You don't need to be a developer to use Redivivus. If you can describe what you want to build, Redivivus can help keep it organized.
